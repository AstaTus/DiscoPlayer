# DiscoPlayer
DiscoPlayer is video player via C++ 11。

#### function
- video play
- change video file
- accurate seek
- play speed
- tune voice

#### platform
- MAC: support
- Window：not test
- IOS: unexpend interface
- ANDROID: unexpend interface

#### MAC depend on:
- ffmpeg: stable 4.2.2
- qt: stable 5.14.1
- sdl: stable 1.2.15
- libYUV: LIBYUV_VERSION 1727
- sound-touch:2.1.2
